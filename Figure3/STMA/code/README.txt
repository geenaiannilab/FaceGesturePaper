order of operations in this directory, to run -- 

1) STMA2021wrapper.m
2) nullSTMAwrapper.m (this is run on server, to take advantage of parallel processing; copy files to Dropbox) 

3) evaluateSTMAwrapper.m (evaluates output of 1 & 2, saves STA to moviefile) 

*BRANCH POINT* -- can either proceed to #4,5 OR directly to #6 OR directly to #7-8 at this point

4) collapseSTMAsAcrossCells.m (generates allCellsSTMAacrossTime_null.mat)
5) plotSTMAsAcrossArrays.m (downloads & plots data from allCellsSTMAacrossTime_null.mat) 

6) analyzeSTMALatencyWholeFrame.m  (generates allCellsVarlatency_(null).mat, which is per cell variance, & mean frame, for all post-spike timebins, within entire frame)

7) generateSTMAbyROI.m (generates faceROIs, saved as 'faceROI_(region).mat', as well as 'allCellsSTMAbyROI_(null).mat', which is per cell variance & mean frame 
for all post-spike timebins, within each of 7 facial ROIs) 

8) analyzeSTMALatencyROI.m (plots per cell, per facial ROI latency metrics)
